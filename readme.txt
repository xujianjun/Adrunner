Display the PriceRunner Adrunner advertisement, both on the slider or the bottom of post.

== Description ==
This is a simple Adrunner WordPress widget for your blog, you can customize the title of box showing the widget.

== Installation ==
1 Download the latest version of the plugin to your computer.
2 With an FTP program, access your site's server.
3 Upload the plugin folder to the /wp-content/plugins folder.
4 In the WordPress administration panels, click on plugins from the menu on the left side.
5 You should see the "Adrunner" plugin listed.
6 To turn the plugin on, click "activate" on the bottom of the plugin name.
7 You should have now a new menu item called "Adrunner Widget" in your widget menu.
